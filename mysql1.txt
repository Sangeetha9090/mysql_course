#whats is relational data model
Relational Model (RM) represents the database as a collection of relations. A relation is nothing but a table of values.
Every row in the table represents a collection of related data values. 
These rows in the table denote a real-world entity or relationship.

#Which is a non relational database?
A non-relational database is a database that does not use the tabular schema of rows and columns found in most traditional database systems. 
Instead, non-relational databases use a storage model that is optimized for the specific requirements of the type of data being stored.

# RDBMS
What is the relational database management system?
The software used to store, manage, query, and retrieve data stored in a relational database is called a 
relational database management system (RDBMS). 
The RDBMS provides an interface between users and applications and the database,
as well as administrative functions for managing data storage, access, and performance.
